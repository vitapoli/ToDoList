# Тест-кейсы для ToDoList приложения

## 1. Аутентификация

| ID | Название | Предусловие | Шаги | Ожидаемый результат | Приоритет |
|----|----------|-------------|------|---------------------|-----------|
| TC-1.1 | Регистрация нового пользователя | Открыта страница входа, пользователь не зарегистрирован | 1. Ввести новое имя и кодовое слово 2. Нажать "Войти" | Пользователь создан, отображается доска задач, в шапке показано имя пользователя | Высокий |
| TC-1.2 | Вход существующего пользователя | Пользователь ранее зарегистрирован | 1. Ввести существующее имя и правильное кодовое слово 2. Нажать "Войти" | Вход выполнен, отображаются ранее созданные задачи | Высокий |
| TC-1.3 | Вход с неверным кодовым словом | Пользователь зарегистрирован | 1. Ввести существующее имя и неправильное кодовое слово 2. Нажать "Войти" | Ошибка, вход не выполнен | Высокий |
| TC-1.4 | Вход с пустым именем | Открыта страница входа | 1. Оставить поле имени пустым 2. Ввести кодовое слово 3. Нажать "Войти" | Форма не отправляется (валидация HTML required) | Средний |
| TC-1.5 | Вход с пустым кодовым словом | Открыта страница входа | 1. Ввести имя 2. Оставить кодовое слово пустым 3. Нажать "Войти" | Форма не отправляется (валидация HTML required) | Средний |
| TC-1.6 | Выход из аккаунта | Пользователь авторизован | 1. Нажать кнопку "Выйти" | Сессия завершена, показана форма входа, localStorage очищен | Высокий |
| TC-1.7 | Сохранение сессии после перезагрузки | Пользователь авторизован | 1. Войти в аккаунт 2. Перезагрузить страницу | Пользователь остаётся залогиненным, задачи загружаются | Средний |
| TC-1.8 | Регистрация с кириллическим именем | Открыта страница входа | 1. Ввести имя на русском языке (например "Полина") 2. Ввести кодовое слово 3. Нажать "Войти" | Пользователь успешно создан, имя корректно отображается | Средний |

## 2. Создание задачи

| ID | Название | Предусловие | Шаги | Ожидаемый результат | Приоритет |
|----|----------|-------------|------|---------------------|-----------|
| TC-2.1 | Создание задачи с названием | Пользователь авторизован | 1. Нажать "Создать +" 2. Ввести только название 3. Отправить | Задача создана, появилась в колонке "Сделать" | Высокий |
| TC-2.2 | Создание задачи со всеми полями | Пользователь авторизован | 1. Нажать "Создать +" 2. Заполнить название, описание, дедлайн, комментарий 3. Отправить | Задача создана со всеми данными, комментарий отображается в разделе "Дополнения" | Высокий |
| TC-2.3 | Создание задачи без названия | Пользователь авторизован | 1. Нажать "Создать +" 2. Оставить название пустым 3. Отправить | Форма не отправляется (поле required) | Высокий |
| TC-2.4 | Создание задачи с длинным названием | Пользователь авторизован | 1. Нажать "Создать +" 2. Ввести название из 500+ символов 3. Отправить | Задача создаётся, текст не обрезается или обрезается корректно | Низкий |
| TC-2.5 | Создание задачи с дедлайном в прошлом | Пользователь авторизован | 1. Нажать "Создать +" 2. Указать дедлайн на вчерашнюю дату 3. Отправить | Задача создаётся (приложение не запрещает прошедшие даты) | Низкий |
| TC-2.6 | Открытие модального окна создания | Пользователь авторизован | 1. Нажать "Создать +" | Модальное окно появляется, фокус на поле названия | Средний |
| TC-2.7 | Закрытие модального окна кнопкой "X" | Открыто модальное окно создания | 1. Нажать "X" | Модальное окно закрывается | Средний |
| TC-2.8 | Закрытие модального окна клавишей Escape | Открыто модальное окно создания | 1. Нажать Escape | Модальное окно закрывается | Средний |
| TC-2.9 | Закрытие модального окна кликом на фон | Открыто модальное окно создания | 1. Кликнуть на затемнённый фон | Модальное окно закрывается | Средний |
| TC-2.10 | Счётчик задач обновляется после создания | Пользователь авторизован, есть колонка "Сделать" | 1. Запомнить число в счётчике "Сделать" 2. Создать задачу | Счётчик увеличился на 1 | Средний |

## 3. Редактирование задачи

| ID | Название | Предусловие | Шаги | Ожидаемый результат | Приоритет |
|----|----------|-------------|------|---------------------|-----------|
| TC-3.1 | Открытие формы редактирования | Существует хотя бы одна задача | 1. Нажать кнопку редактирования (карандаш) на карточке задачи | Форма редактирования появляется с текущими данными задачи | Высокий |
| TC-3.2 | Изменение названия задачи | Открыта форма редактирования | 1. Изменить название 2. Нажать "Сохранить" | Название обновлено на карточке | Высокий |
| TC-3.3 | Изменение описания задачи | Открыта форма редактирования | 1. Изменить описание 2. Нажать "Сохранить" | Описание обновлено | Высокий |
| TC-3.4 | Изменение дедлайна | Открыта форма редактирования | 1. Изменить дедлайн 2. Нажать "Сохранить" | Дедлайн обновлён на карточке | Средний |
| TC-3.5 | Изменение статуса через форму | Открыта форма редактирования | 1. Изменить статус на "В работе" 2. Нажать "Сохранить" | Задача переместилась в колонку "В работе" | Высокий |
| TC-3.6 | Отмена редактирования | Открыта форма редактирования | 1. Изменить данные 2. Нажать "Отмена" | Изменения не сохранены, форма закрылась | Средний |

## 4. Удаление задач

| ID | Название | Предусловие | Шаги | Ожидаемый результат | Приоритет |
|----|----------|-------------|------|---------------------|-----------|
| TC-4.1 | Удаление одной задачи | Существует хотя бы одна задача | 1. Нажать кнопку удаления (X) на карточке задачи | Задача исчезает, счётчик уменьшается | Высокий |
| TC-4.2 | Очистка колонки "Сделать" | В колонке "Сделать" есть несколько задач | 1. Нажать корзину в заголовке колонки "Сделать" 2. Подтвердить удаление | Все задачи из колонки удалены, счётчик = 0 | Высокий |
| TC-4.3 | Очистка колонки "В работе" | В колонке "В работе" есть задачи | 1. Нажать корзину в заголовке колонки 2. Подтвердить удаление | Все задачи из колонки удалены | Средний |
| TC-4.4 | Очистка колонки "Сделано" | В колонке "Сделано" есть задачи | 1. Нажать корзину в заголовке колонки 2. Подтвердить удаление | Все задачи из колонки удалены | Средний |
| TC-4.5 | Отмена очистки колонки | В колонке есть задачи | 1. Нажать корзину в заголовке колонки 2. Нажать "Отмена" в подтверждении | Задачи остаются на месте | Средний |
| TC-4.6 | Очистка пустой колонки | Колонка пустая | 1. Нажать корзину на пустой колонке | Ничего не происходит или показывается сообщение | Низкий |

## 5. Перетаскивание (Drag & Drop)

| ID | Название | Предусловие | Шаги | Ожидаемый результат | Приоритет |
|----|----------|-------------|------|---------------------|-----------|
| TC-5.1 | Перетаскивание из "Сделать" в "В работе" | Есть задача в колонке "Сделать" | 1. Захватить задачу из "Сделать" 2. Перетащить в "В работе" | Задача перемещена, статус обновлён, счётчики обновлены | Высокий |
| TC-5.2 | Перетаскивание в "Сделано" | Есть задача в любой колонке кроме "Сделано" | 1. Захватить задачу 2. Перетащить в "Сделано" | Задача перемещена, done = true | Высокий |
| TC-5.3 | Перетаскивание из "Сделано" обратно в "Сделать" | Есть задача в колонке "Сделано" | 1. Захватить выполненную задачу 2. Перетащить в "Сделать" | Задача перемещена в "Сделать" | Средний |
| TC-5.4 | Визуальная подсветка при перетаскивании | Есть задача для перетаскивания | 1. Начать перетаскивание 2. Навести на колонку | Колонка подсвечивается (класс .drag-over) | Низкий |

## 6. Отметка задачи выполненной

| ID | Название | Предусловие | Шаги | Ожидаемый результат | Приоритет |
|----|----------|-------------|------|---------------------|-----------|
| TC-6.1 | Отметка через кнопку "галочка" | Есть невыполненная задача, вид "Доска" | 1. Нажать кнопку "галочка" на карточке задачи | Задача перемещена в "Сделано", done = true | Высокий |
| TC-6.2 | Отметка через чекбокс в виде списка | Есть невыполненная задача, вид "Список" | 1. Переключиться на вид "Список" 2. Отметить чекбокс напротив задачи | Задача отмечена как выполненная | Высокий |

## 7. Комментарии

| ID | Название | Предусловие | Шаги | Ожидаемый результат | Приоритет |
|----|----------|-------------|------|---------------------|-----------|
| TC-7.1 | Добавление комментария к задаче | Существует задача с развёрнутым блоком "Дополнения" | 1. Развернуть "Дополнения" > "Комментарии" 2. Ввести текст 3. Нажать "Добавить" | Комментарий появился в списке с временной меткой | Высокий |
| TC-7.2 | Добавление нескольких комментариев | Существует задача | 1. Добавить 3 комментария к одной задаче | Все 3 комментария отображаются в правильном порядке | Средний |
| TC-7.3 | Добавление пустого комментария | Существует задача, открыт блок комментариев | 1. Оставить поле комментария пустым 2. Нажать "Добавить" | Комментарий не добавляется | Средний |
