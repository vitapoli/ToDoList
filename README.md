# To Do Board

Веб-приложение для управления задачами в стиле канбан-доски. У каждого пользователя своя личная доска с задачами, данные хранятся в SQLite на сервере.

[Скриншот] (Screenshot_1.png)

## Возможности

- Создание, редактирование и удаление задач
- Колонки: "Сделать" / "В работе" / "Сделано"
- Перетаскивание карточек между колонками (Drag & Drop)
- Табличный вид (список) с фильтрацией по статусу
- Отметка задачи как выполненной
- Комментарии к задачам
- Дедлайны
- Запись голосовых сообщений (микрофон)
- Назначение исполнителей
- Авторизация: имя + кодовое слово
- PWA — можно установить как приложение на компьютер

## Как запустить проект

### 1. Клонировать репозиторий

```bash
git clone https://github.com/vitapoli/ToDoList.git
cd ToDoList
```

### 2. Убедиться что установлен Python 3.11+

```bash
python --version
```

Если не установлен — скачать с https://www.python.org/downloads/

### 3. Запустить сервер

```bash
python server.py
```

### 4. Открыть в браузере

http://localhost:8001

При первом входе с новым именем создаётся личная доска. При повторном входе нужно то же кодовое слово. Другие пользователи не видят чужие задачи.

## Как запустить тесты

### Юнит-тесты (Jest)

```bash
npm install --save-dev jest
npx jest --rootDir . unit-tests.test.js
```

### UI-тесты (Playwright)

```bash
npm install --save-dev @playwright/test
npx playwright install chromium
```

Сначала запусти сервер в одном терминале:

```bash
python server.py
```

Затем в другом терминале запусти тесты:

```bash
npx playwright test ui-tests.spec.js
```

## Технологии

- **Frontend:** HTML, CSS, JavaScript (без фреймворков)
- **Backend:** Python (стандартная библиотека, `http.server`)
- **База данных:** SQLite
- **Тестирование:** Jest (юнит-тесты), Playwright (UI-тесты)
- **PWA:** Service Worker, Web App Manifest
