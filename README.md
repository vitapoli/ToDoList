# To Do Board (серверная версия)

Веб-приложение задач: у каждого пользователя своя доска, данные в SQLite на сервере.

## Что реализовано

- Создание и удаление задач.
- Назначение исполнителя на задачу.
- Колонки: `Сделать` / `В работе` / `Сделано`.
- Перетаскивание карточек между колонками (drag and drop).
- Общий список всех задач (табличный режим).
- Отметка задачи как выполненной.
- Карточка задачи: название, описание, дедлайн, исполнитель.
- Комментарии к задаче.
- Запись голосовых сообщений (микрофон) и воспроизведение внутри карточки.
- Аудио-файлы сохраняются в папку `uploads/` на сервере.
- Пользователи доски (исполнители) внутри личной доски.
- Авторизация: имя + кодовое слово.
- Хранение данных в SQLite (`todo.db`) на сервере.
- PWA-режим: можно установить как приложение на компьютер.

## Структура

- `index.html` - разметка и UI.
- `styles.css` - стили интерфейса.
- `app.js` - логика клиента (работа с API).
- `server.py` - backend API + выдача статики.
- `todo.db` - база данных (создастся при первом запуске).

## Как запустить

### Запуск (локально / на VPS)

```bash
cd todo_app
python server.py
```

Откройте: [http://localhost:8000](http://localhost:8000)

### Установка как приложение (PWA)

1. Запустите сервер (`python server.py`).
2. Откройте приложение в Chrome/Edge по `http://localhost:8000`.
3. В адресной строке нажмите `Установить приложение` (Install App).
4. После установки приложение запускается в отдельном окне, как обычная программа.

## Логика доступа

- При первом входе с новым именем создается новая личная доска.
- При повторном входе требуется то же кодовое слово.
- Другие пользователи не видят задачи этой доски.
